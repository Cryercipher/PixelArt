# 贡献指南

感谢你的贡献！

## 开发设置

```bash
# 克隆并进入项目
git clone https://github.com/Cryercipher/PixelArt.git
cd PixelArt

# 创建虚拟环境
python -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

## 提交流程

1. 创建分支：`git checkout -b feature/xxx`
2. 修改代码
3. 提交：`git commit -m "feat: xxx"`
4. 推送：`git push origin feature/xxx`
5. 提 PR

## 代码风格

- 使用 `black` 格式化
- 遵循 PEP 8
