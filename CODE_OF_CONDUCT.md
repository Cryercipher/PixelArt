# 行为准则

## 我们的承诺

作为本项目的贡献者和维护者，我们致力于为所有人提供一个无骚扰的体验，无论年龄、体形、能力、民族、性别认同和表达、经验水平、教育、社会经济地位、国籍、个人外表、种族、宗教或性身份和取向。

## 我们的标准

有助于为我们社区创造积极环境的行为包括：

* 使用欢迎和包容的语言
* 尊重不同的观点和经验
* 接受建设性的批评
* 专注于对社区最有利的事情
* 对其他社区成员表现出同情

不可接受的行为包括：

* 性骚扰或其他性骚扰行为
* 骚扰、威胁或辱骂
* 个人或政治攻击
* 发布他人的私人信息，例如物理地址或电子邮件地址，未经明确许可
* 其他可能被合理认为在专业环境中不适当的行为

## 执行

项目维护者负责阐明可接受行为的标准，并应对任何不可接受行为的案例采取适当的、公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、问题和其他贡献。

## 范围

本行为准则适用于项目空间内以及当个人代表项目或其社区时的所有空间。代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体帐户发布或作为指定代表在在线或离线活动中行事。

## 执行

可以通过联系项目团队来报告辱骂、骚扰或不可接受的行为。所有投诉将被审查和调查。

项目团队致力于为报告者保密。具体执行政策的进一步细节可能会单独发布。

## 归属

本行为准则改编自[贡献者公约](https://www.contributor-covenant.org)。

有关常见问题，请参阅 https://www.contributor-covenant.org/faq
