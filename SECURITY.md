# 安全政策

## 报告安全漏洞

如果你发现了一个安全漏洞，请**不要**在 GitHub Issues 中报告。相反，请通过电子邮件 security@pixelart.local 报告。

请在报告中包括：
- 漏洞的描述
- 受影响的版本
- 复现步骤（如果可能）
- 潜在影响
- 任何建议的修复（如果有）

我们会尽快确认、评估和修复报告的问题。

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 2.0.x | ✅ 活跃支持 |
| 1.x | ⚠️ 仅关键修复 |
| 0.x | ❌ 不支持 |

## 依赖更新

本项目定期更新依赖项以修复已知的安全问题。建议用户：

1. 定期运行 `pip install --upgrade -r requirements.txt`
2. 启用 GitHub 的依赖项警报
3. 订阅发布通知以接收安全更新
